Documentation

Zanimo is a promise based css transition library and take adventage of the transitionend event which occurs after a transition. The specs defines case where a transition should not work, like trying to make a transition from a state A to the stae A; this implies error handling and Zanimo give you the controle!

Zanimo use the library Q https://github.com/kriskowal/q

Zanimo(elt) ➡ promise

Takes a DOM element and returns a resolved promise of the element.

Zanimo.transition(
    elt,        /* DOM element */
    attr,      /* CSS attribute */
    val,       /* CSS attribute's value */
    t,          /* transition time in milisecond */
    timingF /* timing function */
) ➡ promise

Start a transition

Zanimo.transitionƒ(attr, val, t, timingF) ➡ function

Returns a function wrapping Zanimo.transition wich is conviniant for chaining
